# WindHawk Mods

包含了自己修改的 MOD 以及添加了其他 MOD 的自定义样式

> [!WARNING]
> `windows-11-notification-center-styler` - 1.5 版本有BUG，**ThemeSource** 会有一堆变量失效，请改用 `NotificationCenterStyler1.5.json`

> [!WARNING]
> `StartMenuStyler.json` 在25H2的**(KB5089573)**时会失效并导致菜单栏无法打开（可能是微软把经典布局给删了），请使用 `StartMenuStyler_25h2.json`的最新布局兼容版
